# Movie Recommendation System

This project builds a collaborative filtering movie recommender using rating data. The work focuses on similarity scoring and recommendation logic.

## Tools Used
Python
Pandas
Scikit-learn
Surprise Library

## Notebook Contents
- Data cleaning and transformation
- User and item similarity calculations
- Collaborative filtering model development
- Recommendation generation
- Evaluation using standard metrics

## Key Results
The model produced consistent movie recommendations based on user similarity. Evaluation metrics confirmed the model’s suitability for personalized suggestions.

## Why This Project Matters
This project demonstrates applied machine learning and technician-level recommender system design. It supports personalization and pattern recognition.
