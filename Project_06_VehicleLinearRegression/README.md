# Vehicle Linear Regression

This project builds a regression model using structured automotive data. The work focuses on identifying relationships between vehicle attributes and performance outcomes. The notebook includes data preparation, model development, and evaluation.

## Tools Used
Python
Pandas
Scikit-learn
Matplotlib

## Notebook Contents
- Data cleaning and transformation
- Exploratory analysis of vehicle attributes
- Feature engineering
- Linear regression model development
- Evaluation using standard metrics

## Key Results
The model produced interpretable predictions based on vehicle characteristics. Error metrics confirmed the model’s suitability for basic regression analysis.

## Why This Project Matters
This project demonstrates technician-level statistical modeling and structured data analysis. It supports foundational machine learning skills.
