# Housing Affordability Forecasting – DSC 680 Project 2

This project builds a forecasting model using housing, income, and economic indicators. The work focuses on affordability trends and long-term projections. The notebook includes data ingestion, time series modeling, index construction, and evaluation.

## Tools Used
Python
Pandas
Statsmodels
Matplotlib
Seaborn

## Notebook Contents
- Data collection from Zillow, Redfin, FRED, and Census sources
- Cleaning and alignment of multi-source datasets
- Construction of affordability metrics
- Time series model development
- Forecast visualization and interpretation

## Key Results
The model produced stable affordability forecasts using combined economic indicators. The affordability index showed predictable movement based on interest rates, median income, and median home values.
