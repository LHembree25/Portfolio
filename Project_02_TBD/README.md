# Video Game Sales Predictions – DSC 680 - Project 1

This project builds predictive models using the Video Game Sales with Ratings dataset. The work focuses on identifying relationships between game attributes and sales outcomes. The notebook includes data preparation, feature selection, model development, and evaluation.

## Tools Used
Python  
Pandas  
Scikit-learn  
Matplotlib  
Seaborn

## Notebook Contents
- Data cleaning and transformation  
- Exploratory analysis of game attributes  
- Feature engineering  
- Regression and classification model development  
- Model evaluation using standard metrics

## Key Results
The models produced consistent predictions for global sales based on platform, genre, critic scores, and user scores. Error metrics confirmed the models were suitable for exploratory forecasting and pattern identification.

## Why This Project Matters
This project demonstrates applied machine learning and technician-level modeling skills. It shows the ability to prepare data, build predictive models, and evaluate performance using structured methods.
