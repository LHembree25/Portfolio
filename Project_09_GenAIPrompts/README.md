# Generative AI Prompts and Responses

This project collects structured generative AI prompts and model outputs. The work focuses on evaluating model behavior and documenting prompt design.

## Tools Used
- Python
- OpenAI GPT 4o mini
- Jupyter Notebook

## Notebook Contents
- Prompt design examples
- Model responses
- Evaluation of clarity and consistency
- Documentation of prompt behavior patterns

## Key Results
The prompts produced stable and interpretable model outputs. The evaluation showed predictable behavior across multiple prompt types.

## Why This Project Matters
This project demonstrates applied generative AI experimentation and technician-level prompt engineering. It supports future work in AI-assisted analysis.
