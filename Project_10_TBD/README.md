# Book Recommendation Engine – DSC 680 Project 3

This project builds a collaborative filtering recommendation engine using book ratings data. The work focuses on similarity scoring and personalized recommendations. The notebook includes data preparation, model development, and evaluation.

## Tools Used
Python
Pandas
Scikit-learn
Surprise Library

## Notebook Contents
- Data cleaning and transformation
- User and item similarity calculations
- Collaborative filtering model development
- Recommendation generation
- Evaluation using standard metrics

## Key Results
The model produced consistent recommendations based on user similarity patterns. Evaluation metrics confirmed the model’s suitability for personalized suggestions.

## Why This Project Matters
This project demonstrates applied machine learning and technician-level recommender system design. It shows the ability to build personalized models using structured rating data.
